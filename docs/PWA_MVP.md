# PWA MVP

The first web app is a mobile-first Astro + Preact PWA.

## Included

- State selector.
- Situation selector.
- SIR deadline card.
- Next-action checklist.
- Document checklist.
- Official portal link.
- WhatsApp-shareable checklist.

## Not Included Yet

- Native mobile app.
- User accounts.
- Document uploads.
- National unscoped voter search.
- AI eligibility decisions.
